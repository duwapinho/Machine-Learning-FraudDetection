# Machine-Learning-FraudDetection
ML model to detect fraudulent transactions using dataset of 6.3 mln records. 

Model accuracy: 91%

data i was working on: https://www.kaggle.com/datasets/sriharshaeedala/financial-fraud-detection-dataset/data

Python Libaries i used: 
Pandas, Scikit-learn, Matplotlib, Seaborn, Joblib.
Streamlit for webapp

This project is created for educational and portfolio purposes only and is not intended for commercial use.
